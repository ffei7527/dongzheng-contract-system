## 🚀 部署步骤

### 方法 1：使用 GitHub Actions（推荐）

1. 推送代码到 GitHub：
```bash
git add .
git commit -m "准备部署"
git push
```

2. 在 GitHub 仓库中：
   - 进入 **Settings** > **Pages**
   - Source 选择 **GitHub Actions**
   - 等待构建完成（约 1-2 分钟）

3. 访问网站：
   - `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME/`

### 方法 2：手动部署

1. 构建项目：
```bash
npm run build
```

2. 安装 gh-pages（如果还没有）：
```bash
npm install --save-dev gh-pages
```

3. 部署：
```bash
npm run deploy
```

4. 在 GitHub 仓库设置中选择 `gh-pages` 分支

## ❗ 常见问题

### 页面空白
- 检查浏览器控制台错误
- 确认 `vite.config.ts` 中 `base: './'` 已设置
- 重新构建项目

### 资源 404
- 确认所有路径使用相对路径
- 检查 `dist` 文件夹中的文件
- 清除浏览器缓存

### GitHub Actions 失败
- 查看 Actions 标签页的错误日志
- 确认 `package.json` 中的脚本正确
- 检查 Node.js 版本兼容性

## 📝 更新网站


GitHub Actions 会自动重新部署。
